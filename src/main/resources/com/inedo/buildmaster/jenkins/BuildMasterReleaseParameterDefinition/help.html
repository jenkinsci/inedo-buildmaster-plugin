Defines a parameter where users can pick an active release from a BuildMaster application. The following environment variables will be injected into the build job:

  • BUILDMASTER_APPLICATION_ID: if parameter type is 'Application' or 'Release'
  • BUILDMASTER_APPLICATION_NAME: if parameter type is 'Application' or 'Release'
  • BUILDMASTER_RELEASE_NUMBER: if parameter type is 'Release'
  • BUILDMASTER_LATEST_BUILD_NUMBER: if parameter type is 'Release'
  • BUILDMASTER_NEXT_BUILD_NUMBER: if parameter type is 'Release'
  • BUILDMASTER_BUILD_NUMBER: if parameter type is 'Build'

When used within pipeline script the applicationId parameter can contain either the id or name of a BuildMaster application.

parameters {
  buildMasterRelease applicationId: 'TestApplication', description: '', name: 'BuildMaster Release'
}
