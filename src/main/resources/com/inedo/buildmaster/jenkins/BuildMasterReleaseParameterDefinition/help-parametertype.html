Specifies what the parameter should present at runtime:
  • Application: a list of BuildMaster applications
      Use of this parameter type is optional
  • Release: a list of active releases for the selected BuildMaster application
      This is the only parameter type that must be defined before the variables required for the create build or deploy to stage tasks will be populated
  • Build: a list of active builds for the release selected in the release parameter
      A release parameter must be defined before this parameter can be used
